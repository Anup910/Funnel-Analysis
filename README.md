# E-commerce Conversion Funnel Analysis
# Key Results

1)Analyzed user behavior across an e-commerce conversion funnel to identify where customers drop off in the purchase journey.

2)Measured conversion rates across funnel stages (Homepage → Search → Product View → Purchase).

3)Identified the largest drop-off occurring before the purchase stage, indicating potential friction in the checkout process.

4)Compared desktop vs mobile conversion performance, highlighting opportunities to improve mobile user experience.

5)Generated data-driven recommendations to help improve conversion rates and optimize the product funnel.

# Project Overview

This project performs conversion funnel analysis for an e-commerce website to understand how users move through the purchase journey and where they abandon the process.

A typical customer journey on an e-commerce website follows this path:

Homepage → Search → Product View → Purchase

At each stage, some users leave the website without completing the purchase. Identifying these drop-off points helps product teams optimize the user experience and improve conversion rates.

This analysis demonstrates how data analytics can be used to identify funnel bottlenecks and generate insights for product optimization.

# Business Problem

The company’s leadership observed lower-than-expected sales performance, particularly among new users.

The objective of this analysis is to investigate whether the conversion funnel contains inefficiencies and identify potential areas for improvement.

Key questions addressed in this project:

1)What is the overall funnel conversion rate?

2)At which stage do most users drop off?

3)How does conversion performance differ between desktop and mobile users?

4)What product or UX improvements could increase conversion rates?

# Dataset

The dataset contains user-level interaction events that track progression through the website funnel.

Typical funnel stages analyzed:

1.Homepage Visit

2.Product Search

3.Product Page View

4.Purchase Completion

The dataset includes information such as:

user session activity, funnel stage progression, device type (desktop or mobile), purchase outcomes
